# JVTips Frontend

WEB APP project using Vue 3 and Vite 5.

## Requirements
- [Node](https://nodejs.org/fr/download/package-manager)

## Project Setup

Install dependencies :
```sh
npm install
```

Running application :
```sh
npm run dev
```
